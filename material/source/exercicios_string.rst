
#. Dada a frase ``Python é muito legal.`` use fatiamento para dar nome a
   variáveis contendo cada palavra. O resultado final deve ser:

   .. code::

      >>> frase = "Python é muito legal."
      # resolução do problema aqui
      >>> palavra1
      "Python"
      >>> palavra2
      "é"
      >>> palavra3
      "muito"
      >>> palavra4
      "legal"
